# opencarbon-ruby

A ruby interface to the OpenCarbon disclosure document format.

## Installation

Add this line to your application's Gemfile:

    gem 'opencarbon'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install opencarbon
